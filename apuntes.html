$ php artisan db:seed --class=TasasTableSeeder

Instalar Laravel Collective

// En directorio raiz del proyecto:
$ composer require laravelcollective/html

Permisos y Roles más Usados:

  • givePermissionTo(): nos permite dar seguimiento a un usuario o rol
  • revokePermissionTo(): Revocar el permiso de un usuario o rol
  • hasPermissionTo(): comprueba si un usuario o rol tiene un permiso determinado
  • assignRole(): Asigna rol a un usuario
  • removeRole(): elimina el rol de un usuario
  • hasRole(): comprueba si un usuario tiene un rol
  • hasAnyRole (Role :: all ()): comprueba si un usuario tiene alguna de una lista de roles dada
  • hasAllRoles (Role :: all ()): comprueba si un usuario tiene toda una lista de roles determinada
